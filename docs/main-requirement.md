# 你要解的核心問題
本命題採 Speed Loss（速度損失）分析與油耗預測並行 的方案，包含兩項產出：

## 油耗預測模型（客觀評分） — 給定一艘船在某段航程的航行與環境條件，**預測它當日所使用燃料的主機全速油耗**。同一套模型也應能反事實推論「在現況下執行水下清潔（UWC）或螺旋槳拋光（PP）的節能效益」。這需要你的模型能：
- 從歷史資料中學到「效能如何隨時間變化、又如何在養護後恢復」；
- 理解資料中出現的各類養護／作業事件，並判斷它們各自對船舶運行與油耗的影響方式與程度；
- 在只給部分可見特徵的情況下，推估被遮蔽的油耗數值。

## Speed Loss Dashboard（專家品評） — 於 ISO 19030 框架下計算並視覺化 Speed Loss，呈現三項船務關注重點：
- ① 顯示 Speed Loss（推進效能隨時間的變化趨勢）；
- ② Speed Loss 有多少百分比因船體髒污所致（船殼 vs 螺旋槳的歸因）；
- ③ Speed Loss 與水下清潔資料的比對（效能恢復或惡化的時間點是否與養護事件對應）。

> 提示：資料中有一類事件屬於「純檢查、無實體介入」——它不清洗、也不拋光。聰明的模型應能辨識出這類事件不應帶來效能改善，而非一律把「養護後」當成「效能變好」。
