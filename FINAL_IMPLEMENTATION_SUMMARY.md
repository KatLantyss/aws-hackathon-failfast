## 🎯 關鍵技術創新

### 1. 斜率基礎異常檢測 ⭐⭐⭐

**原理**：用 Speed Loss 曲線的斜率變化判定維修效果，而非單純中位數差值

```
維修前斜率 slope_before = +0.050 %/day（衰退）
維修後斜率 slope_after = -0.020 %/day（改善）
─────────────────────────────────────────
斜率改善 = 0.050 - (-0.020) = 0.070 %/day ✓ 優秀

評估：維修成功阻止了衰退！
```

**優勢**：
- 不受短期波動影響
- 能區分「臨時現象」vs「真實改善」
- 更符合海事運營邏輯

**驗證結果**（74 個維修事件）：
- 異常率：44.6% → 32.4% ⬇ 12.2%
- 成功率：55.4% → 67.6% ⬆ 12.2%
- 方法一致性：45.9%（穩定可靠）

### 2. 維護類型特定的異常判定 ⭐⭐⭐

**原理**：不同維修類型有不同的成功率和期望改善值

| 類型 | 成功率 | 期望改善 | 異常閾值 |
|------|--------|---------|---------|
| DD | 85.7% | -13.27% | 斜率 > 0.15 |
| UWI+PP | 51.6% | -4.45% | 斜率 > 0.06 |
| PP | 63.6% | -3.04% | 斜率 > 0.04 |
| UWC | 16.7% | +2.23% | 斜率 > 0.04 |
| UWI | 0% | 0% | 任何改善異常 |

### 3. 經濟驅動的優先級排序 ⭐⭐

**公式**：`Priority Score = Speed Loss × 預期月度節油 USD`

**意義**：優先處理「污損嚴重 × 節油價值高」的船舶

**例**：
- S11: SL 42%, 月度節油 USD $22,750 → 優先級高
- S5: SL 8%, 月度節油 USD $2,000 → 優先級低
