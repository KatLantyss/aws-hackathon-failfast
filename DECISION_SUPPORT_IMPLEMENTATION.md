## 🎯 決策支持邏輯

### 維修建議算法

```
IF 當前 Speed Loss ≥ 30%:
  建議 → DD（進塢）
  理由 → 船體污損嚴重，需全面處理
  成功率 → 85.7%

ELSE IF 當前 Speed Loss ≥ 20%:
  建議 → UWC+PP（複合維修）
  理由 → 中度污損，複合維修應有效
  成功率 → 57.1%

ELSE IF 當前 Speed Loss ≥ 10%:
  建議 → UWC 或 PP（單項維修）
  理由 → 輕度污損
  成功率 → 50-60%

ELSE:
  建議 → 監控中
  理由 → 船體狀況良好
  成功率 → N/A
```

### ROI 計算公式

```
月度節省 = 預期節油 (MT/月) × 燃油價格 ($/MT) × 成功率

年度節省 = 月度節省 × 12

回本週期 (月) = 維修成本 / 月度節省

ROI% = ((年度節省 - 維修成本/12) / 維修成本) × 100
```
